# Smart Micro-Job Platform

## Project Description
This project is a responsive web application that helps users find short-term micro-jobs quickly and easily.

## Problem
Many students and freelancers struggle to find small temporary jobs efficiently.

## Solution
This platform provides a simple interface to browse, filter, and explore job opportunities with interactive features.

## Target Users
- Students
- Freelancers
- Job seekers

## Main Features
- Dynamic job rendering using JavaScript
- Live search filtering
- Job details view
- Contact form validation
- Dark mode toggle
- Bootstrap table styling
- jQuery animation effects
- Responsive design

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- jQuery
- Font Awesome

## How to Run
1. Download the project.
2. Open index.html using Live Server in VS Code.
3. Explore all sections and features.
